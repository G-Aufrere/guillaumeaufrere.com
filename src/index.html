A propos

Développeur front-end depuis 2 ans

Je suis un développeur frontend originaire de Bourges, disponible pour des missions freelance ou en contrat CDI.

Mon but est de créer des sites web interactifs et accessibles sur tous les supports.

J'aime vraiment manipuler les éléments sur une page web, les rendre vivants grâce aux animations, que ce soit en 2D ou en 3D. J'éspère rendre cette intérêt visible sur mes prochains projets. Créer un site web attractif est désormais devenu une priorité pour moi.

Je travaille principalement en Javascript pur, néanmoins je suis actuellement en train de m'orienter sur Vue.js.
Pour les animations j'utilise GSAP.
